Refreshing expired assets

Asset URLs expire after ~7 days. Use the Figma MCP tools to get fresh URLs: 

// For bottom nav icons:
get_design_context(
  fileKey: "BdJG2rVEiEQJFeKsvxSLm7",
  nodeId:  "27287:46066"
)

// For any ZenDS icon by name:
search_design_system(
  query:           "Icon-CardMCOutline",
  fileKey:         "7TTXh09o7SDNDDFBZWK3WC",
  includeLibraryKeys: ["lk-d5da8e..."]
)

// For wallet bubbles or other screen assets:
get_design_context(
  fileKey: "BdJG2rVEiEQJFeKsvxSLm7",
  nodeId:  "<node id from Figma URL>"
)

Always include a fallback onerror SVG on every <img> tag so screens remain functional even after asset URLs expire.
